# 데미안 (Demain)

`데미안`은 `나의 위대한 사서`입니다. PDF 파일을 제시하면, RAG 기반으로 요약합니다.

</br>

1. yarn 설치

   ```bash
   yarn install
   ```

2. expo 시작

   ```bash
    npx expo start
   ```
